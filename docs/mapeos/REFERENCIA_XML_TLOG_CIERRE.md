# Estructura XML de Salida — TLOG_CIERRE

## Reglas de generación

- **INICIO DE TURNO / INICIO DE PERÍODO** → solo cabecera (sin `<ItemList>`)
- **FIN DE TURNO** → cabecera + detalle
- **FIN DE PERÍODO** → solo cabecera (sin `<ItemList>`)

---

## Cabecera

```xml
<?xml version="1.0" encoding="utf-8"?>
<Transaction>
    <RETAILSTOREID>KOSTST.KST_CODE</RETAILSTOREID>
    <WORKSTATIONID>0</WORKSTATIONID>
    <SEQUENCENUMBER>construir (Bridge, 11 dígitos, por tipo de documento)</SEQUENCENUMBER>
    <BUSINESSDAYDATE>nombre del archivo formato AAAA-MM-DD</BUSINESSDAYDATE>
    <BEGINDATETIME>BusinessDayDate + BEGIN_DATE_OFFSET (config)</BEGINDATETIME>
    <ENDDATETIME>BusinessDayDate + END_DATE_OFFSET (config)</ENDDATETIME>
    <OPERATORID>admin</OPERATORID>
    <PERIODO>0</PERIODO>
    <SUBPERIOD>0</SUBPERIOD>
    <PERIODCODE></PERIODCODE>
    <SUBPERIODCODE></SUBPERIODCODE>
    <TYPECODE>BusinessEOD</TYPECODE>
    <TYPEID>63</TYPEID>
```

---
